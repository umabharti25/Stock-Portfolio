 # 📈 Stock Portfolio Tracker

A simple Python application that allows users to build a stock portfolio, calculate the value of their investments, and optionally save the portfolio summary to a text file.

## 🚀 Features

- Add stocks to a portfolio using stock symbols.
- Enter the quantity of shares owned.
- Calculate the value of each stock holding.
- Display the total investment value.
- Validate stock symbols and quantities.
- Save the portfolio summary to a text file.
